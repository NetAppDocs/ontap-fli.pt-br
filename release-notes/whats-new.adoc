== ONTAP 9.19.1

* A partir do ONTAP 9.19.1, o ONTAP FLI oferece suporte ao protocolo iSCSI para conectividade de backend.




== ONTAP 9.17.1

* A partir do ONTAP 9.17.1, os sistemas ASA R2 são suportados para migração FLI.
+
Saiba mais sobre link:https://docs.netapp.com/us-en/asa-r2/get-started/learn-about.html["Sistemas ASA R2"^] .
