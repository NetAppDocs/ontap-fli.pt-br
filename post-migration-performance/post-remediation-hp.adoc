== Pós-Migração - FLI Online

Após excluir a importação do LUN, verifique se o LUN está configurado com  `-space-reserve disabled` e  `-space-allocation enabled`. Caso contrário, modifique-o de acordo.

Para que o recurso SCSI UNMAP funcione corretamente, consulte a documentação do fornecedor para verificar as versões de software compatíveis. A combinação Veritas VxVM + VxFS é necessária para habilitar esse recurso.

[cols="4"]
|===
| Sistema de arquivos | Dispositivo de bloco | LVM (FS) | Sanboot 


| Etapa 1: execute uma nova verificação de armazenamento em todos os hosts HP-UX mapeados | Etapa 1: execute uma nova verificação de armazenamento em todos os hosts HP-UX mapeados. | Etapa 1: execute uma nova verificação de armazenamento em todos os hosts HP-UX mapeados | Etapa 1: execute uma nova verificação de armazenamento em todos os hosts HP-UX mapeados 
|===
.Passos
. Identificar dispositivos de armazenamento.
+
[source, cli]
----
ioscan -fnC disk
insf -e
sanlun lun show all
sanlun lun show all -pv
sanlun lun show -wwpn
----
. Importe VG e coloque o VG online.
+
[source, cli]
----
vgimport /dev/vg_name /dev/disk/disk_name
vgchange -a y /dev/vg_name
----
. Identifique o grupo de volume e os sistemas de arquivos.
+
[source, cli]
----
vgdisplay -v /dev/vg_name
vgdisplay /dev/vg_name
lvdisplay /dev/vg_name/lv_name
----
. Monte o sistema de arquivos.
+
[source, cli]
----
bdf
mount /dev/vg_name/vol_name /mount_point
----
